# Tienda Mania

1. Clonar Proyecto
2. Instalar dependencias
```
yarn install
```
3. Clonar el archivo __.env.template__ y renombrarlo a __.env__
4. Configurar las variables de entorno
5. Levantar la base de datos
```
docker-compose up -d
```
